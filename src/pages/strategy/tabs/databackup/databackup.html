    Backup v2

    On starting the v2 backup process, you will be asked to select the backup folder. Then, you will be asked for permission to use the folder.

  • Backup all data
  • Update current backup
  • Import data

    • Read the WARNINGS first!

    • DO NOT go to other tabs while importing your data.
    • DO NOT close the window while importing your data.
    • DO NOT input other files.
    • Especially large files may take over an hour to finish. BE PATIENT.

    This is an experimental backup feature that provides more stability at the cost of speed. It also allows for updating the current backup without having to perform a full backup. Currently, it is only supported on Chrome v86 and higher. Please note that the v2 backup files do not with with v1 backup and vice versa.

Please be patient.
This process may take a long time.
DO NOT TOUCH ANYTHING
BEFORE THE "Finished!" MESSAGE!!!
